# jCourse Client
## 基于Qt的选课社区客户端
### 1.简介
一个基于Qt的跨平台选课社区客户端，对标[选课社区网站](https://course.sjtu.plus)开发，同时增加了存档机制，实现社区内容的本地访问。
### 2.开发目的
* 通过存档机制，分担选课高峰期流量，减轻服务器压力
* 通过本地化访问社区资源，提升高峰期的用户选课体验
### 3.基本功能
* 选课社区基础功能
  - [x] 检索课程
  - [x] 查看课程评价
  - [ ] 发表课程评价
  - [ ] 回应课程评价
* 拓展功能
  - [x] 社区资源的自动缓存
  - [ ] 社区资源的手动缓存
  - [ ] 缓存资源的自动化管理
