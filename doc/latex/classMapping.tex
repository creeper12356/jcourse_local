\doxysubsection{Detailed Description}
包装类，表示教师和课程的关系 

Mapping类用于表示教师与课程的关系，如果教师\+A（使用\+Teacher类对象teacher\+A表示）讲授课程\+B（使用\+Course类对象course\+B表示）， 那么这种关系可以表示为\+Mapping(\&teacher\+A,\&course\+B)。
